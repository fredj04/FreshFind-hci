Heuristic Evaluation

Nielsen's 10 Usability Heuristics — peer evaluation findings and design responses

Evaluation Setup

Evaluating team: FoodPath (peer team, community nutrition mapping project)

Method: Independent review of all three task flows using Nielsen's 10 Usability Heuristics. Issues recorded on index cards with heuristic violated and severity rating (0=no problem, 4=catastrophe). Findings consolidated in group discussion.

10 Issues Found

#HeuristicIssueSev.
H11 — Visibility of System StatusNo loading indicator on map fetch — blank screen for 2–4 seconds on slow connections3
H21 — Visibility of System StatusTimestamp "22 min ago" ambiguous — time zone and meaning of 'verified' unclear2
H33 — User Control & FreedomNo back button on language screen during first launch2

Design Changes Made in Response

C1H1 · Sev. 3

Added skeleton-screen animation on map load — pulsing gray placeholder circles appear at pin locations while live data fetches.

C2H2 · Sev. 2

Timestamp now reads "Community confirmed open · 22 min ago" with an info icon that opens a tooltip explaining the verification model.

C3H3 · Sev. 2

Back/Skip button added to language selection screen. On first launch with no previous state, back defaults to "Skip — continue in English."

C4H4 · Sev. 2

Unified color system: green=open, yellow=closing, orange=out of stock, gray=closed, red=error only. Task 3 confirmation uses green for action success (labeled "Update submitted" to distinguish from resource state).

C5H5 · Sev. 2

Submit button enlarged (44→52px). Cancel moved to top-sheet text link, spatially separated from Submit to prevent fat-finger errors.

C6H6 · Sev. 2

Active filter pills: filled green background + white text + checkmark icon. Inactive: white background + gray border. Contrast ratio now exceeds 4.5:1. Filled vs. outlined distinction supports colorblind users.

C7H7 · Sev. 2

Added floating action button (FAB) on map view for quick 2-tap contribution. FAB chosen over dedicated tab to keep contribution secondary to discovery in navigation hierarchy.

C8H8 · Sev. 1

Listing card redesigned with 4 visual tiers using progressive disclosure: (1) name+status, (2) hours+distance, (3) SNAP+timestamp, (4) notes+actions.

C9H9 · Sev. 3

Undo promoted to outlined button with 5-minute countdown: "Undo available for 4:32." Timer sets clear expectations and reduces anxiety about accidental submissions.

C10H10 · Sev. 1

First-use tooltip on timestamp: "Community members keep this updated — tap to learn more." Shown once on first listing view, then dismissed and preference saved.
